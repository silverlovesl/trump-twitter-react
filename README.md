# How to use

## Installation

```sh
yarn install
```

## Add .env.local file on your local

```sh
touch .env.local
# set follow lines
REACT_APP_API_ENDPOINT=http://localhost:5000
REACT_APP_COOKIE_TIMEOUT=8
REACT_APP_IS_USE_MOCK=true #set false to connection api
```

## Start up

```sh
yarn start # default port 8081
```
